## 核心功能

- 实体化坐标标点（无碰撞体积，不影响游戏行为）
- 屏幕方向指引系统（实时指向目标坐标）
- 多标点管理与基础坐标解析
- 轻量级架构设计，便于扩展 UI / 数据同步
- 模块化设计，支持后续功能扩展

---

## 游戏内能力

- 在世界中创建“不可碰撞标记实体”
- 自动追踪目标坐标方向
- 屏幕实时显示导航指示
- 支持多标点切换与管理
